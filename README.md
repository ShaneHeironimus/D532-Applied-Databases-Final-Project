# D532-Applied-Databases-Final-Project
Indiana University D532-Applied Databases Bloomington, IN animal shelter adoption dashboard final project.

This project is designed to execute the various topics covered throughout the semester. Key aspects include
creating a relational database and normalizing the data from a single file.

The project consists of two parts, the database_normalization.py and the app.py. The first file takes the raw
CSV data and creates the respective tables for normalization. It also inserts the records into the respective
tables to be queried later. The app.py handles the UI/UX portion and is where the bulk of the action happens.
This is where users can perform the various CRUD operations on the dashboard.
